Description

A quiz game with a clean and intuitive UI, featuring 10 random history questions about Portugal. Players earn 10 points for each correct answer and lose 10 points for each incorrect one. Survivor's Legacy

Play Here

Description

A top-down 2D pixel art game where players must progress through three levels, each filled with enemies that must be defeated to unlock the exit. The player can switch between three different weapons, collect items, and use two special powers to survive. Developed using Unity Engine, the game incorporates the new input system for seamless control. Enemy pathfinding includes radius detection to track the player's movements. Additionally, it utilizes Unity's Universal Render Pipeline (URP) with lighting effects and basic post-processing to enhance the visual quality.

The Brave

Play Here

Description

A 2D pixel art RPG featuring crafting, talent trees, and equipment systems. The game takes place in a single level where the player gathers resources, gains experience to learn new skills, and defeats enemies in preparation for the final battle against the King. Developed using Unity Engine, the game implements a StateMachine for both player and enemy behaviors, along with a Save and Load system using JSON for data storage.
